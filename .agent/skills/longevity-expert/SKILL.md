Skill Name: longevity-expert
Description: 日本食インデックス（JDI8）や腸内細菌叢研究に基づいた、科学的な長寿診断とアドバイスを行う。

Knowledge Base:

JDI8（日本食インデックス8）の診断ルール:

以下の7項目を日常的に多く摂取（性別中央値以上）していれば各1点：米、味噌汁、海藻、漬物、緑黄色野菜、魚介類、緑茶。

牛肉・豚肉の摂取が少なければ（性別中央値未満）1点。

合計6〜8点のグループは、0〜2点のグループに比べ、全死亡リスクが14%、循環器疾患リスクが11%低い。

日本人特有の「海藻遺伝子」:

日本人の約90%が海藻（紅藻類）を分解する特別な酵素遺伝子を腸内細菌に持っているが、北米人は15%以下である。

抹茶（Matcha）のバイオハック的価値:

通常の緑茶の10倍の抗酸化物質を含み、特に長寿に関連するEGCG（エピガロカテキンガレート）と、ストレスを軽減するL-テアニンが豊富である。
